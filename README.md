# London to Dakar Rally: Trip Planner

## TL;DR
*   **Goal**: Drive from London (UK) to Banjul (The Gambia) in 21 days.
*   **Dates**: ~Jan 16, 2026 -- Feb 6, 2026.
*   **Vehicles**: Budget cars (destined for auction/charity in Gambia).
*   **Route**: UK -> France -> Spain -> Morocco -> Western Sahara -> Mauritania -> Senegal -> The Gambia.

## Repository Structure

This repo contains the consolidated guide built from the official Roadbook and US State Department advisories.

### `/docs`
The core documentation.
*   **Country Guides**:
    *   `europe-transit.md`: France & Spain transit (Logistics, Kit).
    *   `morocco.md`: Ferries, Atlas Mountains, Western Sahara politics.
    *   `mauritania.md`: *Critical* (E-Visa, Alcohol ban, Landmines).
    *   `senegal.md`: The Diama crossing, Car Pass, Zebrabar.
    *   `the-gambia.md`: The Finish Line, Car Handover.
*   **Planning & Prep**:
    *   `visas.md`: Visa requirements, costs, and border procedures.
    *   `car-prep.md`: Mandatory kit (Sand ladders, Radios), Spares, and "The Buck" admin.
    *   `personal-kit.md`: Packing for 3 climates (Snow to Desert).
    *   `_todos.md`: **Active Checklist** of tasks (Admin, Purchases, bookings).
    *   `_roadbook.txt`: The raw source material (Reference only).

### `/plans`
Agentic planning documents used to generate the documentation. (Archival use).
