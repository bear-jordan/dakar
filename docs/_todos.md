# Trip Action Items

## Immediate Priority
- [ ] **Passport Check**: Ensure passport is valid for at least 6 months beyond the planned return date.
- [ ] **Vaccinations**: Get Yellow Fever vaccination and ensure you have the **Yellow Card** certificate (Required for Mauritania/Senegal).
- [ ] **Mauritania**: Apply for E-Visa online (do not wait for the border).
- [ ] **Gibraltar/UK**: Apply for UK ETA (Electronic Travel Authorisation) if traveling after Jan 8, 2025.

## Documentation to Print
- [ ] **Fiches**: Print ~50-100 copies of the "Fiche" (personal/car info sheet) for checkpoints.
- [ ] **Passport**: Colour copies of main page.
- [ ] **Car Documents**: Colour copies of V5/Registration.
- [ ] **Driving Licence**: Colour copies.
- [ ] **Insurance**: Physical copies of green cards/insurance docs.

## Vehicle & Logistics
- [ ] **Senegal**: Secure "Senegal 1 Car Pass" (contact organizers/Dave).
- [ ] **Cash**: Withdraw/Exchange Euros (small denominations) for border fees (Mauritania/Senegal/Gambia).
- [ ] **Photos**: Bring extra passport-style photos for border forms (The Gambia/Mauritania).

## 3-4 Weeks Before Departure
- [ ] **Register**: Enroll in US State Department STEP program for alerts.
- [ ] **Maps**: Download offline maps (Maps.me / Google Maps) for all transit countries.
